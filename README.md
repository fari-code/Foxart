# 🚀 Foxart - SaaS Landing Page

Une landing page SaaS moderne et responsive développée avec React et Tailwind CSS.  
Ce projet propose une interface propre et élégante, une mise en page fluide, un design sombre moderne ainsi qu’une responsivité optimisée pour les startups et les produits technologiques.

---

## ✨ Fonctionnalités

- 🎨 Interface moderne et professionnelle
- 📱 Responsive sur mobile, tablette et desktop
- ⚡ Développé avec React + Vite
- 💨 Stylisé avec Tailwind CSS
- 🌙 Design dark mode moderne
- 🧩 Composants réutilisables
- 🚀 Performance rapide et optimisée

---

## 🛠️ Technologies utilisées

- React.js
- Tailwind CSS
- Vite
- JavaScript

---

## 📸 Aperçu

Landing page moderne conçue pour les startups SaaS, applications web, services tech ou produits digitaux.
